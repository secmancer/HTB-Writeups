# Modules
- A **module** is the core learning unit in HTB Academy, representing a single course on a specific topic.
- Each module contains multiple **sections**, and several modules together form a **path**.
- On the **Modules** page, you can preview a module’s summary to see:
    - What the module covers
    - The expected learning outcomes
    - Any prerequisites or requirements before starting

> **Note:** You can click on the heart icon to add a module to your favorite "i.e. `To-Do`" list, which can be accessed from the [Dashboard](https://academy.hackthebox.com/dashboard).


# Types of Module
- HTB Academy modules fall into three categories: **Offensive**, **Defensive**, and **General**.
- Each module is rated by difficulty: **Fundamental**, **Easy**, **Medium**, or **Hard**.
- Modules can be filtered by category, difficulty, or other criteria on the **Modules** page.
- You can also search by module name to quickly find specific content.

# Module Tiers
- Modules are split into `5 Tiers` based on their cost, as follows:

| Tier   | Cost | Reward |
| ------ | ---- | ------ |
| Tier 0 | 10   | 10     |
| Tier 1 | 50   | 10     |
| Tier 2 | 100  | 20     |
| Tier 3 | 500  | 100    |
| Tier 4 | 1000 | 200    |
- Each module requires a specific number of **cubes** to unlock.
- Completing a module 100% returns **20%** of the cubes spent.
- **Tier 0 modules** are effectively free: they cost 10 cubes and return 10 upon completion.
- Cubes are refunded only after full completion to encourage finishing modules.
- A module’s **Tier** does **not** indicate its difficulty level.

> **Note:** If you have an annual subscription, then included modules will cost you 0 cubes, while still giving you back their reward.


# Completing a Module
- To complete a module, you must:
    - Solve all exercises
    - Mark all sections as **completed**
- Completion usually unlocks a **Skills Assessment**, simulating a real-world scenario to test your understanding.
- Passing the Skills Assessment confirms mastery of the module’s content.
- Click **Finish** (typically in the Skills Assessment section) to finalize completion.
- After finishing, you’ll be taken to the **Completion page**, where you can:
    - Share your achievement on social media
    - Write a review    
    - View suggested next modules
    - Unlock and view the module’s **badge** on the **My Badges** page.
