# What is a Path
- **Paths** are collections of modules focused on a broader topic.
- Some paths culminate in an **HTB Certification** after completing the associated exam.
- A **single module** can appear in multiple paths if its content aligns with different topics.


# Types of Paths
- HTB Academy offers two types of paths: **Skill Paths** and **Job-Role Paths**.
- You can **enroll** in any path by clicking the **Enroll** button.
- Enrolling helps you:
    - **Track progress** within the path.
    - **View upcoming modules** directly from the **Dashboard** page.

> **Note:** Enrolling in a path does not unlock all of the modules within it. Hence, enrolling in a path does not cost any cubes.

- **Skill Paths** group modules focused on learning a specific, in-demand skill.
    - Example: _Operating System Fundamentals_ path teaches OS basics.
- **Job-Role Paths** are designed to prepare learners for specific cybersecurity roles.
- Each **Job-Role Path** is tied to an **HTB Certification**, earned by passing a **multi-day practical exam**.

> **Tip:** You can estimate how long a path will take to complete by checking the duration mentioned underneath the path's title. This represents the sum of the `estimated time for completion` of each module in the path "in 8-hour work days".


# Obtaining a Certification
- An **HTB Certification** is the main achievement goal in HTB Academy.
- Each certification is tied to a **Job-Role Path**, designed to prepare you for its exam.
- Learners of all levels can progress through modules and paths to earn a certification.
- Earning a certification requires completing:
    - **Hundreds of sections and exercises**
    - **Dozens of modules and Skills Assessments**
    - A **multi-day real-world practical exam**
- Completing these steps ensures you are **job-ready** for the targeted cybersecurity role.

> **Note:** Completing a path also unlocks a new badge, and so does obtaining a certificate.
