# Wireless Network
- Communication system using wireless signals, like radio waves
- Allows for connectivity, sharing of files, accessing services, or other things over the air
- Advantages
	- Mobility
	- Ease of installation
	- Scalability
- Disadvantages
	- Interference
	- Security risks
	- Speed limitations


# Wireless Router
- Device that forwards data packets between networks
- Often, a wireless router does both routing and Wireless Access Point


# Terms
- WAN (Wide Area Network) Port: connects to your Internet source (e.g., a cable modem)
- LAN (Local Area Network) Ports: for wired connections to local devices (e.g., desktop computer, printer).
- Antennae: transmit and receive wireless signals. (Some routers have internal antennae)
- Processor & Memory: handle routing and network management tasks


# Mobile Hotspot
- A smartphone can share its cellular internet connection over Wi-Fi, allowing devices like laptops and tablets to connect. It works over 4G/5G networks, has a short range, consumes significant battery power, and is usually password-protected.


# Cell Tower
- A cell tower provides cellular network coverage for a specific geographic area (cell). Phones connect to nearby towers for calls and data. Towers are managed by Base Station Controllers (BSCs), which handle handoffs between towers as users move. Towers connect to the core network through fiber optic or microwave backhaul links.
	- **Macro cells:** Large towers covering wide rural areas.
	- **Micro/Small cells:** Smaller installations used in dense urban environments to improve coverage.


# Wireless Frequencies
- Wireless communication relies on radio waves measured in hertz (Hz).
	- **2.4 GHz Wi-Fi:** Longer range and better wall penetration but more interference.
	- **5 GHz Wi-Fi:** Faster speeds but shorter range.
	- **Cellular Frequencies:** Range from low bands (e.g., 700 MHz) for broad coverage to high bands (e.g., 28 GHz+) for high-speed 5G.
	- **Frequency Trade-Offs:**
		- Lower frequencies = longer range, less data capacity.
		- Higher frequencies = shorter range, greater data capacity.
		- Shared frequencies can become congested, causing interference.
		- Government agencies (e.g., FCC) regulate frequency allocation to prevent interference and ensure efficient spectrum use.
